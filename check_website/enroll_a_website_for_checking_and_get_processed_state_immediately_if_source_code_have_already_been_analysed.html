Check Website API

Enroll a website for checking and get processed state immediately if source code have already been analysed

POST /api/v1/sites

Send to the server page URL, its content and all JS

Parameters

Name Description
key Unique client key
site[url] Website URL to check for malicious scripts
site[scripts_attributes] Web page source code (with all JS) to make ckeck of
site[scripts_attributes][][name] Script name
site[scripts_attributes][][type] Script type can by one of "html" or "javascript"
site[scripts_attributes][][content] Script content

Request

Headers

Content-Type: application/json

Route

POST /api/v1/sites

Body

{"key":"57736ace6a122f44e241dc2a","site":{"url":"http://example.com","scripts_attributes":[{"name":"Page Content","type":"html","content":"\u003chtml\u003econtent\u003c/html\u003e"},{"name":"http://example.com/javascripts/jqery.js","type":"javascript","content":"1234"}]}}

cURL

curl "http://localhost:9292/api/v1/sites" -d '{"key":"57736ace6a122f44e241dc2a","site":{"url":"http://example.com","scripts_attributes":[{"name":"Page Content","type":"html","content":"\u003chtml\u003econtent\u003c/html\u003e"},{"name":"http://example.com/javascripts/jqery.js","type":"javascript","content":"1234"}]}}' -X POST \
	-H "Content-Type: application/json"

Response

Headers

Content-Type: application/json
Content-Length: 105
Vary: Origin
X-Content-Type-Options: nosniff

Status

201 Created

Body

{
  "type": "sites",

  "attributes": {
    "url": "http://example.com",
    "state": "safe"
  }
