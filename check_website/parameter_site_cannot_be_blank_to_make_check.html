Check Website API

Parameter site cannot be blank to make check

POST /api/v1/sites

If provided new site attributes are not present server will return an error.

Parameters

Name Description
key Unique client key
site[url] Website URL to check for malicious scripts
site[scripts_attributes] Web page source code (with all JS) to make ckeck of
site[scripts_attributes][][name] Script name
site[scripts_attributes][][type] Script type can by one of "html" or "javascript"
site[scripts_attributes][][content] Script content

Request

Headers

Content-Type: application/json

Route

POST /api/v1/sites

Body

{"key":"1234","site":{}}

cURL

curl "http://localhost:9292/api/v1/sites" -d '{"key":"1234","site":{}}' -X POST \
	-H "Content-Type: application/json"

Response

Headers

Content-Type: application/json
Content-Length: 85
Vary: Origin
X-Content-Type-Options: nosniff

Status

400 Bad Request

Body

{
  "message": "Parameter cannot be blank",
  "errors": {
    "site": "Parameter cannot be blank"
  }
}
