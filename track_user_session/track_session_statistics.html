Track User Session API

Track session statistics

GET /api/v1/sessions/:key

Server tracks all URLs visited by a particular user. Users can see a number of safe and not safe sites that they visited.

Parameters

Name Description
key User session unique key

Request

Route

GET /api/v1/sessions/57736ace6a122f44e241dc34

cURL

curl "http://localhost:9292/api/v1/sessions/57736ace6a122f44e241dc34" -X GET

Response

Headers

Content-Type: application/json
Content-Length: 116
Vary: Origin
X-Content-Type-Options: nosniff

Status

200 OK

Body

{
  "type": "sessions",
  "id": "57736ace6a122f44e241dc34",
  "attributes": {
    "safe_sites_count": 1412,
    "malicious_sites_count": 2
  }
}
